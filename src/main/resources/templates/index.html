System Overview

The portal supports the core tutoring workflow used by students and tutors during routine academic support operations.

  • Students can review active tutoring slots and submit bookings for available times.
  • Tutors can add new availability, adjust scheduled sessions, and retain cancelled records for reference.
  • Shared status and feedback messages keep session activity clear across each page.
